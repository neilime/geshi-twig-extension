üì¢ **Geshi Twig Extension** is a [Twig](https://twig.symfony.com) extension for [GeSHi - Generic Syntax Highlighter rendering](https://github.com/GeSHi/geshi-1.0).

```twig
{{ '<?php\necho \'test\';\n?>' | geshi('php') }}
```

```twig
{% geshi 'javascript' %}
    {"data": "test"}
{% endgeshi %}
```
